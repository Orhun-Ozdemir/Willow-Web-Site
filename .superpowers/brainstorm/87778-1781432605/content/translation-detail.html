✨ "Tümünü Çevir" nasıl çalışır?

İngilizce metni yazdıktan sonra butona basınca boş olan tüm diller otomatik dolar. Çevirileri tek tek onaylayabilir ya da düzenleyebilirsin.

1

Google Translate API

Ücretsiz: günde 500.000 karakter. Kalite iyi, kurulumu basit. Ücretsiz limit aşılınca küçük ücret var (çok düşük).

2

DeepL API

Ücretsiz: ayda 500.000 karakter. Avrupa dilleri için çok daha kaliteli çeviri. TR desteği sınırlı.

3

Claude API (Mevcut)

Zaten Anthropic hesabın var. En kaliteli + bağlam anlayan çeviri. Kısa metinler için maliyet ihmal edilebilir (1000 çeviri = ~$0.10).
